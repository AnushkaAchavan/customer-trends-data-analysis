# 👩🏻‍💻 Customer Behavior Data Analysis Project

A complete **end-to-end Data Analytics portfolio project** demonstrating real-world business problem solving using **Python, SQL, and Power BI**.

This project simulates how data analysts work in corporate environments — transforming raw customer data into actionable business insights and stakeholder-ready dashboards.

---

## 📌 Project Objective

The objective of this project is to analyze customer shopping behavior data and generate meaningful insights that help businesses:

- Understand customer segments
- Identify purchasing patterns
- Analyze revenue drivers
- Improve customer retention strategies
- Enable data-driven decision-making

---

## 🔄 End-to-End Workflow

This project follows a structured analytics pipeline:

### 1️⃣ Business Understanding
- Define business goals
- Identify key performance indicators (KPIs)

### 2️⃣ Data Modeling & EDA (Python)
- Import dataset
- Data cleaning and preprocessing
- Handle missing values
- Exploratory Data Analysis (EDA)
- Feature preparation
- Export cleaned data to SQL

### 3️⃣ Data Analysis (SQL)
- Create relational database
- Load cleaned dataset into SQL
- Write analytical queries
- Perform aggregations, filtering, grouping, segmentation
- Answer business-driven questions

### 4️⃣ Dashboard Development (Power BI)
- Connect SQL database to Power BI
- Create data model
- Design interactive dashboard
- Build KPI cards, trend charts, segmentation visuals

### 5️⃣ Reporting & Presentation
- Summarize findings
- Provide business recommendations
- Prepare stakeholder-ready presentation

### 6️⃣ Version Control
- Organize project files
- Maintain proper documentation
- Host project on GitHub
